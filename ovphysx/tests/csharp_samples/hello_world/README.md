# Hello World C# Sample

Creates a PhysX instance, loads a USD scene, and runs one simulation step.

This is the C# port of `tests/c_samples/hello_world_c/main.c`.

## Build and Run

```bash
cd ovphysx/tests/csharp_samples/hello_world
dotnet run
```

## Requirements

- .NET 8 SDK
- ovphysx native library in the system library path
